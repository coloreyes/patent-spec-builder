# 专利说明书生成 — 入口指令

## 用途

启动专利说明书生成流程。根据用户提供的技术交底书或论文，生成标准专利说明书格式的 Word 文档（说明书摘要 + 说明书）。

## 流程

### 步骤 1：确认输入源

检查用户是否已提供：
- 技术交底书路径（.md / .docx），或
- 技术论文路径（.pdf / .docx / .md）

若未提供，询问用户：
```
请提供技术交底书或论文的路径，我将将其转换为标准专利说明书格式（含说明书摘要和说明书）。
```

### 步骤 2：读取输入内容

根据文件类型读取：
- `.md`：直接 Read
- `.docx`：使用 `python-docx` 读取
- `.pdf`：使用 `docx_to_md.py` 或 `python-docx`（如为原始 Word 导出）

### 步骤 3：执行转换

**Read** `prompts/spec_builder.md`，严格按照其中的：
- 输出格式要求
- 撰写风格规范
- 从交底书到说明书的映射规则
- Word 生成技术要求

完成以下操作：
1. 分析交底书结构，提取关键内容
2. 按映射规则重组为说明书格式
3. 生成说明书摘要（单段，200-400字）
4. 生成完整说明书（含 S1-Sn 步骤、公式、系统模块、有益效果、实施例）
5. 输出两份 Word 文档

### 步骤 4：输出文件

按命名规则输出：
```
{案件名}_{YYYYMMDDHHmmss}-说明书摘要.docx
{案件名}_{YYYYMMDDHHmmss}-说明书.docx
```

### 步骤 5：确认交付

向用户报告：
- 生成的文件路径
- 说明书包含的章节数、步骤数、公式数、实施例数
- 摘要字数
